# Отчёт

StudentNumber | Номер студенческого билета   
LastName | Фамилия   
FirstName | Имя   
MiddleName | Отчество  
BirthYear | Год рождения   
Grade1, Grade2, Grade3 | Три оценки   
Поле для вычисления: | Средняя оценка считается через (Grade1 + Grade2 + Grade3) / 3 

---

### Формат хранения данных в файле
Текстовый файл `.csv`, поля разделены `;`  
Пример строки:
10352;Ivanov;Ivan;Petrovich;2003;9;8;10


Есть два режима:
перезапись файла
добавление новых записей в конец файла

---

### Сортировка и индексация
Реализованы два индекса:

| Индекс | Ключ | Сортировка | Алгоритм |
|--------|------|-------------|----------|
| IndexByLastName | Фамилия | По возрастанию | Merge Sorting |
| IndexByAverage | Средняя оценка | По убыванию | Bubble Sorting |

---

### Поиск
Реализованы два варианта бинарного поиска по индексам:

| Тип поиска | Ключ | Реализация |
|------------|------|------------|
| Итерационный | Фамилия | По упорядоченному индексу LastName |
| Рекурсивный | Средняя оценка | По индексу Average |

При обнаружении совпадения отображается полная запись студента.

---

### Дополнительно

Редактирование записи: Выполнено изменение данных и перестройка индексов
Логическое удаление: Выполнено отметка DeletedMask
Физическая очистка массива: Выполнено функция PackArray() удаляет помеченные записи
